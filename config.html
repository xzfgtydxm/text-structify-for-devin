提示词模板管理

创建不同的提示词模板，用于不同的整理风格。在插件弹窗中可快速切换当前模板。

提示：提示词会影响 AI 如何整理你的文字。不同模板可针对不同场景优化。

快捷键

默认未绑定快捷键。你可以自行设置：

前往 chrome://extensions/shortcuts 为「整理当前输入框的文字」设置快捷键

使用方法

  1. 点击插件图标 → 配置 API 端点（Base URL + API Key）→ 选择 LLM 模型
  2. 在网页上用自带的语音输入（或手动输入）写好文字
  3. 点击输入框旁边的 ✨ 整理 按钮，或按 Ctrl+Shift+1
  4. AI 自动读取页面对话上下文，将文字整理成结构化内容
  5. 整理后的文字直接替换回输入框

支持的 API 端点

  • OpenAI — api.openai.com
  • DeepSeek — api.deepseek.com
  • Groq — api.groq.com（免费额度，速度快）
  • Ollama — localhost:11434（本地模型）
  • 任何 OpenAI 兼容 API — 自定义 Base URL
